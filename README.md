# Crypto Portfolio

## Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/crypto-portfolio.git
   cd crypto-portfolio
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm start
   ```

4. **Open the application in your browser**

   Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Usage

To use Crypto Portfolio, simply navigate through different pages to view cryptocurrency prices, track your portfolio, and explore various features.
